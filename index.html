Shwe Kone Thal Mobile Application အသုံးပြုသူများ လိုက်နာရမည့် စည်းမျဉ်းနှင့် သတ်မှတ်ချက်များ

Shwe Kone Thal Application မှတစ်ဆင့် ရွှေရောင်းဝယ်မှု ဝန်ဆောင်မှုအား တောင်းခံခြင်း သို့မဟုတ် အသုံးပြုခြင်းသည် ဤစည်းမျဉ်းနှင့် သတ်မှတ်ချက်များကို လက်ခံအသိအမှတ်ပြုခြင်းဖြစ်သည်။

၁။ အဓိပ္ပာယ်ဖွင့်ဆိုချက်များ

အောက်ဖော်ပြပါ အဓိပ္ပာယ်အသုံးအနှုန်းများသည် ဤစည်းမျဉ်းနှင့် သတ်မှတ်ချက်များတွင် အသုံးပြုသောအခါ အောက်ပါအတိုင်း အဓိပ္ပာယ်ဖွင့်ဆိုမည်ဖြစ်သည်။

၁.၁ “Account” ဆိုသည်မှာ အသုံးပြုသူများအား Shwe Kone Thal တွင် ဝယ်ယူထားသော ရွှေများကို သိမ်းဆည်းထားနိုင်စေရန် Shwe Kone Thal မှ ပံ့ပိုးပေးသော အစီအစဉ်တစ်ခုဖြစ်သည်။

၁.၂ “ကိုယ်စားလှယ်” ဆိုသည်မှာ Shwe Kone Thal ၏ ကိုယ်စား လုပ်ငန်းဆောင်ရွက်မှု တစ်စိတ်တစ်ဒေသကို Shwe Kone Thal နှင့် သဘောတူညီချက်ရယူကာ ဆောင်ရွက်သော ကြားခံဝန်ဆောင်မှုပေးသူကို ဆိုလိုပါသည်။

၁.၃ “သဘောတူညီချက်” ဆိုသည်မှာ Shwe Kone Thal မှ ထုတ်ပြန်ထားသော အသုံးပြုသူစာရင်းသွင်းမှုပုံစံ အပါအဝင် ဤစည်းမျဉ်းနှင့် သတ်မှတ်ချက်များအားလုံးကို ဆိုလိုပါသည်။

၁.၄ “အလုပ်ဖွင့်ရက်” ဆိုသည်မှာ မြန်မာနိုင်ငံရှိ စနေနေ့၊ တနင်္ဂနွေနေ့နှင့် အများပြည်သူ အလုပ်ပိတ်ရက်များကို မပါဝင်သောနေ့ရက်များကို ဆိုလိုပါသည်။

၁.၅ “Wallet Account” ဆိုသည်မှာ Shwe Kone Thal မှ ပံ့ပိုးပေးထားသော Digital Wallet Account ဖြစ်ပြီး အသုံးပြုသူများအား အီလက်ထရွန်နစ်စနစ်ဖြင့် ရွှေများ သိမ်းဆည်းခြင်းနှင့် လွှဲပြောင်းမှုများ ပြုလုပ်နိုင်စေရန် ရည်ရွယ်ထားသော Account ဖြစ်သည်။

၁.၆ “အခကြေးငွေ” ဆိုသည်မှာ Shwe Kone Thal မှ ပေးအပ်သော ဝန်ဆောင်မှုများ အသုံးပြုရာတွင် ကောက်ခံသည့် အခကြေးငွေအားလုံးကို ဆိုလိုပါသည်။

၁.၇ “Shwe Kone Thal” ဆိုသည်မှာ လက်ရှိ Shwe Kone Thal Mobile Application နှင့် အနာဂတ်တွင် ပံ့ပိုးပေးမည့် ဝန်ဆောင်မှုများအားလုံးကို ဆိုလိုပါသည်။
